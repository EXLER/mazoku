## Requirements

* Python 3.7
* [Pyxel 1.2.10](https://github.com/kitao/pyxel)

## Usage
```bash
# Install dependencies
$ pip install -r requirements.txt

# Run the game
$ python3 mazoku.py
```

## Controls

<p align="center">
    <img width="568" src="docs/screen.png">
</p>

* `W/S` - Move forward/backward
* `A/D` - Turn left/right
* `M` - Turn minimap off/on
* `R` - Generate new maze
